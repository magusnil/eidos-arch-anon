## ICML 2026 Reproduction Scripts

This folder contains paper-specific Dataset 1 runners used for the ICML 2026 submission and follow-up table regeneration.

Files:

- `train_dataset1_v7_mlp_baselines.py`: Table 1 and Table 2 baselines
- `train_dataset1_v7_table3.py`: Table 3 no-PathBundle ablation
- `train_dataset1_v7_table5.py`: Table 4 and Table 5 PathBundle scorer runs
- `train_dataset1_v7_pathbundle.py`: earlier paper-specific harness retained for comparison and debugging

The generic quick-start demos remain in `public_release/examples/`.
